MultiCheck

MultiCheck is jquery plugin which convert your multiple select in set of checkboxes

Basic usage

HTML

<select multiple="multiple">
  <option value="red">Red</option>
  <option value="green">Green</option>
  <option value="blue">Blue</option>
</select>

JavaScript

$('select').multicheck();

Bootstrap style

HTML

<select multiple="multiple">
  <option value="red">Red</option>
  <option value="green">Green</option>
  <option value="blue">Blue</option>

$('select').multicheck({
  label_wrap: '<div class="checkbox"></div>'

  <option data-label-class="label label-success" value="blue">Blue</option>
